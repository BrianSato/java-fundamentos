# ☕ Curso de Java

Este repositório reúne os exercícios, exemplos e projetos desenvolvidos durante meus estudos da linguagem **Java**, acompanhando o curso **Java COMPLETO - Programação Orientada a Objetos + Projetos**, ministrado por **Nélio Alves** na plataforma **Udemy**.

O objetivo deste repositório é registrar minha evolução na linguagem Java, consolidando os conceitos por meio de exercícios práticos e servindo como referência para consultas futuras.

---

## 🎯 Objetivos

- Aprender os fundamentos da linguagem Java.
- Desenvolver uma base sólida em Programação Orientada a Objetos (POO).
- Aplicar os conceitos em exercícios práticos.
- Manter um histórico da minha evolução durante os estudos.
- Preparar a base necessária para o desenvolvimento com Spring Boot.
